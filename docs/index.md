## Overview

`@ardinsys/financial-charts` is a low-level canvas charting engine focused on financial time series.
It provides primitives to render candles, bars, area charts, and custom controllers while remaining lightweight.

Use the Guide section to integrate the library into your project, then browse the Reference for the complete API.
